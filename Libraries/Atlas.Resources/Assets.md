When adding a new file:
- Use 24 x 24 png files
- Add in the Assets folder
- Embed the image in the project (Alt-Enter on file, select Embedded)
  - Properties
	- Build Action
	  - Embedded
- Add with git
  - git ignore these files for some reason
  - "git add -f add.png"
  - How to fix .gitignore
	- `!*.png` doesn't work
